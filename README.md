# API Security Assurance Harness (Swagger Petstore)

This repository documents a small, end-to-end API security workflow:
spec scanning (KICS) → runtime tests (Postman) → automation (Newman) → manual validation (Burp) → final report.

Target:
- Base URL: https://petstore3.swagger.io/api/v3
- OpenAPI:  https://petstore3.swagger.io/api/v3/openapi.json
